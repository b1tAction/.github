# b1tAction

b1tAction 是一个围绕 **Paradice / ParaDiced** 的小游戏项目组织，当前主要维护游戏前端、后端服务与官网展示站点。

## 主要仓库

- [paraweb](https://github.com/b1tAction/paraweb)：ParaDiced Web 客户端，使用 TypeScript 构建游戏前端体验。
- [paradiced](https://github.com/b1tAction/paradiced)：ParaDiced 游戏后端，基于 Go + Nakama + CockroachDB 提供权威对局与服务能力。
- [website](https://github.com/b1tAction/website)：组织官网（[bitaction.cn](https://bitaction.cn)）代码仓库，用于品牌与项目信息展示。
